One Dimensional Search

What is One Dimensional Search

一维搜索的理论是建立在整个最优化理论之下的。

最优化算法的基本理论是：

  1. 求下降方向\(p_k\)，使得\(\nabla f(x_k) p_k < 0\)
  2. 求取步长\(\alpha_k\)，使得\(\alpha_k\)，使得\(f(x_k + \alpha_k p_k) < f(x_k)\)
  3. 返回迭代，直至序列\(\{x_k\}\)收敛。

一维搜索是指选择合适的\(\alpha_k\)，使得 $$ f(x_k + \alpha_k p_k) = \min \limits_{\alpha > 0} f(x_k + \alpha p_k) $$

按这种方法确定的步长因子叫做最优步长因子。这种方法的想法是，既然函数的下降方向已经确定了，为何不沿着这个方向找到使目标函数取极小的点呢。

而且，这种想法也是容易办到的，因为\(f(x_k + \alpha p_k)\)是以\(\alpha\)为变量的一元函数\(\phi(\alpha)=f(x_k+\alpha p_k)\)。

例如： 假设函数\(f(x,y) = x^2+y^2+2xy\)在\((1,1)\)点已经确定了方向\((2,3)\)是下降方向，那么一维搜索是要找到合适的\(\alpha\)，使得 \(\phi(\alpha)=f(1+2\alpha,1+3\alpha)=(1+2\alpha)^2+(1+3\alpha)^2+2(1+2\alpha)(1+3\alpha)\)的函数值最小。

求一元函数极小点的方法称为直线搜索或者一维搜索，许多最优化的问题都把多维问题转为一维问题。因此可以说一维搜索是最优化方法的一个重要支柱

  • 这种方法的优点是：它使函数值在搜索方向上下降得最多
  • 缺点是：计算量较大

The Type of One Dimensional Search

一维搜索分为精确搜索和不精确搜索两类。

  • 精确搜索：为了在搜索方向上下降得最多，所以要精确找到最优的步长。
    • Golden Section Method
    • Parabolic Interpolation
  • 不精确搜索：当迭代点的距离离极小点较远时，精确求解最优步长因子对问题没有太大的意义。
    • Inexact One Dimensional Search

确定搜索区间

精确搜索和不精确搜索都是建立在函数搜索区间是单谷函数的基础上，如果函数不是单谷函数，那么要进行搜索区间的确定。
